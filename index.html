🤖 IA para analizar nóminas

La app usa Gemini (gratis) para analizar PDFs. Introduce tu API key de Google AI Studio. Si tienes API key de Anthropic también puedes añadirla — se usará como preferencia.

🟢 Gemini API Key (gratuito · aistudio.google.com)
🔵 Anthropic API Key (de pago · opcional)
🟠 Groq API Key (gratuito · console.groq.com)
Alternativa gratuita a Gemini. Muy rápida. Crea tu key en console.groq.com → API Keys.
Orden de prioridad:
1º Anthropic (si tienes key y funciona)
2º Gemini (gratuito)
3º Groq (gratuito · alternativa)
Las keys se guardan solo en este dispositivo, nunca en la nube.
